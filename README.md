# Information 
 - When Maintenance Mode is Active, Only Players With OP or The Permission "maintenancemode.bypass" Can Join Your Server.
 - Maintenance Mode Kick Message is Configurable Via The Conig File.
# Commands
- /mm on
- /mm off
# Permissions
